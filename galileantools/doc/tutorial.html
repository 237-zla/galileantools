Galileantools tutorial

Introduction

This vignette demonstrates how to simulate a projectile launched from a moving platform, apply Galilean transformations, and estimate the platform velocity from noisy measurements using the galileantools package.

Load package

library(galileantools)
set.seed(42)

## Simulation

# We simulate a projectile launched from a platform moving at v = 15 m/s.

dat <- simulate_projectile(u0 = c(5,12), pos0 = c(0,1.5),
t = seq(0,2.565,0.05), v_platform = 15, noise_sd = 0.2)
